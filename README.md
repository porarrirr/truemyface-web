# TrueMyFace Web

TrueMyFace の公開用ウェブページ（GitHub Pages）。アプリ DesignSystem 準拠のネオブルータル濃紺 UI。

- `index.html` — ランディング
- `support.html` — サポート / FAQ
- `privacy.html` — プライバシーポリシー
- `terms.html` — 利用規約

## ローカルプレビュー

```sh
cd Web
python3 -m http.server 8765
```

http://localhost:8765/index.html を開いて確認してください。
